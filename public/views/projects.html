Projects

Here you can find a list of the projects I've worked on. For more in-depth information you can check out my GitHub profile where I publish most of my code.

  • My LoL Fantasy
    JavaScript, HTML5, CSS3, SQL
    • Web app I made with my friends at a hackathon
    • The app tracks you and your friends' stats in League of Legends games and compares them in a Fantasy-esque system
    • ExpressJS web server renders jade templates for client requests
    • User and statistic data is persisted in a PostgreSQL database
    • Redis is used to cache session data and Riot Games API requests
  • Personal Website
    JavaScript, HTML5, CSS3
    • Website I created to showcase my professional career and to gain some web development skills
    • Uses the following frameworks: Node.js, AngularJS, jQuery
  • Briscola Stat Tracker
    Java
    • A simple app I made to delve into Android development
    • It keeps stats of bids and points earned for the trick-based card game Briscola
  • Project Euler and HackerRank
    C++, Python
    • I like puzzles so I have done a number of problems on these sites
    • 30 problems solved on Project Euler
    • ~260 points earned in Algorithms on HackerRank
  • Pokémon Type Effectiveness Table
    Python, Regular Expression
    • Out of frustration with new Pokémon being released and the new Fairy type, I wanted a quick lookup table for which moves to use against which Pokémon
    • Parses the HTML of a web page on Bulbapedia with a Regular Expresion
    • A Python script uses the parsed data to compose a table of type effectivenesses against every Pokémon
  • Minesweeper
    C++
    • I wanted to automatically keep track of some more specific stats while playing Minesweeper so I created my own app for it
    • Full game complete with graphics, settings, and persisted statistics
    • Works on Windows, OS X, and Linux
    • OS specific tasks are done using SFML
  • Dominion
    C++
    • I created a text-based version of this deck-building card game because shuffling decks is annoying
    • Every feature and card of the original version of the game is implemented
